# Blog Veterinaria

Este proyecto es un blog estático para documentar el desarrollo iterativo e incremental de un sistema web para una veterinaria. El blog está diseñado con un enfoque minimalista, profesional y responsive, ideal para proyectos universitarios.

## Estructura del contenido
- **index.html**: Página principal
- **marco-teorico.html**: Marco teórico
- **analisis-estructurado.html**: Análisis estructurado
- **analisis-orientado-objetos.html**: Análisis orientado a objetos
- **planificacion.html**: Planificación
- **contactos.html**: Contactos

## Diseño
- Colores: verde veterinaria (#4CAF50), blanco, gris claro
- Fuente: Arial, sans-serif
- Navegación clara y responsive

## Funcionalidades
- Menú de navegación adaptable
- Galería de imágenes para diagramas
- Fechas en publicaciones
- Botones para compartir en redes sociales

## Personalización
Todos los archivos incluyen comentarios para facilitar la edición y adaptación a las necesidades del grupo.
